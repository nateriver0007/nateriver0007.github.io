如果要生成几百万个唯一ID呢？

几百万个ID去判断是否重复，开销太大，只能想办法在生成的时候就是唯一的，对于这个问题python标准库中已经有了解决方法，UUID

UUID是128位的全局唯一标识符，通常由32字节的字符串表示。它可以保证时间和空间的唯一性，它通过MAC地址、时间戳、命名空间、随机数、伪随机数来保证生成ID的唯一性。
